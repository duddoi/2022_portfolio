1. 회원가입 및 로그인

1.1 연구원 로그인

국제지재권 분쟁예방 컨설팅의 전문기관 소속 연구원들의 로그인 방식에 대해 안내합니다

① 회원가입 때 작성한 휴대폰 번호를 이용한 로그인 방법입니다. 연구원의 이름, 휴대폰 번호, 인증코드를 입력하면 로그인이 가능합니다.

② 연구원 정보에서 휴대폰 번호를 수정하기 전, 휴대폰 번호가 변경되었을 경우 Switch 기능을 통해 이메일로 로그인이 가능합니다. 해당 버튼을 클릭하면 이메일 로그인 페이지로 전환 됩니다.
* 휴대폰 번호, 이메일이 연구원 정보에서 수정하기 전, 변경된 경우 시스템 담당자에게 연락하여 주시기 바랍니다. (시스템 담당자 번호 : 070 - 4470 – 3452)
